# ECO6130 Machine Learning in Finance - Group Project

This repository contains the code and data files for the ECO6130 group project.

## Project files

- `01_construct_input_window_and_5d_labels.ipynb`  
  Constructs 20-day input windows and 5-day binary labels using CSI300 data.

- `02_baseline_revised_schemeA.ipynb`  
  Implements baseline machine learning models and Scheme A backtesting.

- `03_rendering_comparison_schemeA.ipynb`  
  Compares different image rendering schemes for the 2D CNN model.

- `04_rendering_detail_changes_schemeA.ipynb`  
  Examines detailed rendering changes under Scheme A.

- `CSI300_merged_2005_2026.csv`  
  Source CSI300 market data used in the notebooks.

- `CSI300_window20_label5d.npz`  
  Preprocessed OHLC rolling-window dataset generated from Notebook 01.

- `CSI300_window20_label5d_ohlc_volume_ma.npz`  
  Preprocessed OHLC + volume + MA20 rolling-window dataset used in Notebooks 03 and 04.

## How to run

1. Clone or download this repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Open the notebooks in Jupyter Notebook or JupyterLab.
4. Run the notebooks in order:

```text
01_construct_input_window_and_5d_labels.ipynb
02_baseline_revised_schemeA.ipynb
03_rendering_comparison_schemeA.ipynb
04_rendering_detail_changes_schemeA.ipynb
```

The notebooks use relative paths, so they should be run from the root folder of this repository.

## Notes

- The original local absolute paths have been changed to relative paths for reproducibility.
- Model checkpoint files generated during training are not included in the repository.
- If the preprocessed `.npz` files are removed, they can be regenerated by running the relevant notebooks.
